# Recall Command

Use this command to search the Cerebro vault and return the most relevant notes. {{ $ARGUMENTS }}

## Behavior

- Search the vault first.
- Prefer Atlas/Dots and Atlas/Maps.
- Return paths and a short synthesis.
- Do not invent content.

## Input

The user query.

## Output

- Direct answer
- Relevant source paths
- Optional follow-up question if the query is ambiguous
