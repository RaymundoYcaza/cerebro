# Atajos principales para plegado (Folds)

## Notas
- **`za`**: Alternar pliegue actual (colapsar si está abierto, abrir si está cerrado).
- **`zc`**: Colapsar (Close) el pliegue actual.
- **`zo`**: Abrir (Open) el pliegue actual.
- **`zM`**: Cerrar todos los pliegues (Close all).
- **`zR`**: Abrir todos los pliegues (Open all).
