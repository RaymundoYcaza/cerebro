### 🧠 Observaciones útiles

- Para construir ordinales **compuestos** (ej. 267, 467), se combinan los ordinales correspondientes a las centenas, decenas y unidades:
    
    - Ejemplo: 467 = _cuadringentésimo sexagésimo séptimo_
        
- Del 1000 en adelante, se utiliza el patrón **[número cardinal] + "milésimo"**.
    
- Si el número es redondo (sin decenas o unidades), solo se usa el ordinal base: _quingentésimo_ (500), _tresmilésimo_ (3000).
    
- Para documentos o listas formales, usa el género que coincida con el sustantivo al que modifica (ej. _la página quingentésima_, _el capítulo milésimo_).
