## Buscar un texto en nano
Para realizar una búsqueda en el editor de texto `nano`, puedes seguir estos pasos:
Abre el archivo en `nano` con el comando `nano nombre_del_archivo`.
Una vez que el archivo esté abierto, presiona `Ctrl + W`. [Esto abrirá el campo de búsqueda](https://askubuntu.com/questions/47515/any-way-to-search-for-text-within-nano)[1](https://askubuntu.com/questions/47515/any-way-to-search-for-text-within-nano).
Escribe el texto que estás buscando y presiona `Enter`. [El cursor de `nano` se moverá a la primera coincidencia del texto buscado](https://askubuntu.com/questions/47515/any-way-to-search-for-text-within-nano)[1](https://askubuntu.com/questions/47515/any-way-to-search-for-text-within-nano).
[Si deseas buscar la siguiente coincidencia del texto, puedes presionar `Alt + W`](https://askubuntu.com/questions/47515/any-way-to-search-for-text-within-nano)[1](https://askubuntu.com/questions/47515/any-way-to-search-for-text-within-nano)[2](https://fedingo.com/how-to-search-in-nano-text-editor-in-linux/).
[Si deseas buscar la coincidencia anterior del texto, puedes presionar `Alt + Q`](https://askubuntu.com/questions/47515/any-way-to-search-for-text-within-nano)[3](https://devicetests.com/search-text-nano-editor)
