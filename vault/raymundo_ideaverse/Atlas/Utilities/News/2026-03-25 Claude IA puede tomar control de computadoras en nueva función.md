# Claude IA puede tomar control de computadoras en preocupante nueva función

- Agencia: [[Montevideo Portal]]
- url: https://www.msn.com/es-xl/noticias/other/claude-puede-tomar-control-de-computadoras-en-preocupante-nueva-funci%C3%B3n/ar-AA1Zp3VG

## 📌 Resumen de la noticia
- Anthropic lanzó una función experimental en Claude que permite a la IA controlar directamente computadoras.
- Claude Code puede abrir archivos, usar navegadores y ejecutar tareas como si fuera un usuario humano.
- El sistema utiliza “connectors” para integrarse con aplicaciones, pero cuando no están disponibles recurre al control directo del escritorio.
- Expertos advierten riesgos de privacidad, ya que la IA debe capturar la pantalla y acceder a permisos amplios.
- La función está en fase de “research preview”, disponible solo en Mac y para usuarios pagos.

## 🔑 Puntos principales
- Claude Code introduce control directo de computadoras mediante IA.
- Uso de “connectors” como método más seguro frente al acceso total a la pantalla.
- Riesgos: captura de documentos, chats y datos sensibles.
- Requiere permisos de accesibilidad y grabación de pantalla.
- Limitaciones: lento, impreciso y con alto consumo de recursos.
- Disponible únicamente para usuarios Pro y Max en Mac.
- Anthropic reconoce que las salvaguardas no son absolutas.
