## 🧠 Resumen (Español)

Este libro proporciona una introducción accesible desde el punto de vista matemático a las herramientas necesarias para abordar problemas de inferencia modernos en ingeniería y ciencia de datos. Está diseñado para estudiantes de posgrado que toman cursos de inferencia estadística y detección/estimación, y sirve como una referencia valiosa para investigadores y profesionales. El libro cubre temas como pruebas de hipótesis, estimación y análisis de rendimiento, con numerosos ejemplos y problemas para reforzar la comprensión del lector.

## 🧠 Summary (English)

This book provides an accessible mathematical introduction to the tools needed to address modern inference problems in engineering and data science. It is designed for graduate students taking courses in statistical inference and detection/estimation, and serves as a valuable reference for researchers and professionals. The book covers topics such as hypothesis testing, estimation, and performance analysis, with numerous examples and problems to reinforce the reader’s understanding.

## 🎯 Ideas Clave

- Bayesian and Frequentist approaches to statistical inference
- Hypothesis testing techniques (Neyman-Pearson, Minimax)
- Estimation methods (Maximum Likelihood, Bayesian)
- Performance analysis of decision rules
- Detection theory and signal detection
- Applications in engineering and data science
