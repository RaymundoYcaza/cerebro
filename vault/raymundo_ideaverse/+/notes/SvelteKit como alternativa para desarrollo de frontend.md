Svelte es un framework para desarrollar rápidamente robustas aplicaciones web usando Svelte.

Svelte es un ==framework de JavaScript de código abierto que se utiliza para construir interfaces de usuario==. A diferencia de otros frameworks como React o Vue, Svelte no realiza la mayor parte de su trabajo en el navegador mientras la aplicación se ejecuta, sino que traslada este trabajo a un paso de compilación durante el desarrollo. Esto resulta en un código JavaScript altamente optimizado y ligero.
