Shadcn/ui es una ==biblioteca de componentes de interfaz de usuario (UI) diseñada para React, construida sobre Radix UI y Tailwind CSS==. Se enfoca en la accesibilidad, la facilidad de personalización y la integración con flujos de trabajo de desarrollo modernos. Ofrece componentes reutilizables y fáciles de usar para crear interfaces de usuario consistentes y atractivas.
