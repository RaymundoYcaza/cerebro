Qué Es El MERN Stack Y Qué Lo Compone?
- El stack MERN es un conjunto de tecnologías utilizado para desarrollar aplicaciones web completas utilizando JavaScript tanto en el lado del cliente (frontend) como en el lado del servidor (backend). El acrónimo "MERN" significa: MongoDB, Express, React y Node.js.
