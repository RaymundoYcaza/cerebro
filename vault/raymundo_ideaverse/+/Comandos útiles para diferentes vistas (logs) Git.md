# Solo títulos (una línea)
```bash
git log --oneline
```

# Solo títulos con más contexto
```bash
git log --pretty=format:"%h %s" -10
```

# Log completo (lo que viste)
```bash
git log
```

# Log con estadísticas de archivos
```bash
git log --stat
```
# Log gráfico con ramas
```bash
git log --graph --oneline
```
