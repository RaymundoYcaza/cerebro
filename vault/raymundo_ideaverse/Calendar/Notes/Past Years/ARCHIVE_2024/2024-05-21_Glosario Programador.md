	- ## Interpolar variables
	  collapsed:: true
		- En programación, especialmente en **JavaScript**, interpolar variables a menudo se refiere a la inserción de valores de variables dentro de cadenas de texto. [Esto se hace comúnmente usando **plantillas literales** en ECMAScript 6 (ES6), donde puedes usar marcadores de posición para inyectar variables en una cadena](https://www.freecodecamp.org/espanol/news/como-utilizar-la-interpolacion-de-cadenas-en-javascript/)[2](https://www.freecodecamp.org/espanol/news/como-utilizar-la-interpolacion-de-cadenas-en-javascript/). Aquí tienes un ejemplo de cómo se vería en JavaScript:
		- **JavaScript**AI-generated code. Review and use carefully. [More info on FAQ](https://www.bing.com/new#faq).
		- ```
		  const nombre = "Mundo";
		  const saludo = `¡Hola, ${nombre}!`;
		  console.log(saludo); // Imprime: ¡Hola, Mundo!
		  ```
		- En este ejemplo, la variable `nombre` se interpola dentro de la cadena de texto que está entre las comillas invertidas, y el resultado es una cadena que incluye el valor de la variable.
