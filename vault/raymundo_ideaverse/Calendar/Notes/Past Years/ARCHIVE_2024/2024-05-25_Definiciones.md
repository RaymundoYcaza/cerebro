Library - a small or medium piece of software that you can include in your code to do certain things easily instead of writing them from ground up.
Framework - A huge software, where you define / write your code in their way, agreeing to their prior set of terms, to create a piece of software.
Package - A bundle of code, that you can include / import in your code and use it for your needs.
