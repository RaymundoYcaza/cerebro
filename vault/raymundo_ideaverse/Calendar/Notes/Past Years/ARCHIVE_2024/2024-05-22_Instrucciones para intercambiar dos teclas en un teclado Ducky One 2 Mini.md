



 **Activación del modo de intercambio de teclas**: Mantén presionadas las teclas `Fn` + `Alt` + `k` durante 5 segundos. Sabrás que has ingresado al modo de intercambio de teclas cuando estas se iluminen con colores.
 **Selección de las teclas a intercambiar**: Una vez en el modo de intercambio de teclas, presiona las dos teclas que deseas intercambiar.
 **Finalización del proceso**: Para salir del modo de intercambio de teclas y guardar los cambios, presiona la tecla `Esc`.
**Cambiar la tecla fn con los switches físicos**

| DIP1 | DIP2 | Configuration (from left to right) | Notes                |
| ---- | ---- | ---------------------------------- | -------------------- |
| OFF  | OFF  | alt, windows, fn, ctrl             | This is the default. |
| ON   | OFF  | fn, alt, windows, ctrl             |                      |
| OFF  | ON   | alt, fn, windows, ctrl             |                      |
| ON   | ON   | alt, windows, ctrl, fn             |                      |
