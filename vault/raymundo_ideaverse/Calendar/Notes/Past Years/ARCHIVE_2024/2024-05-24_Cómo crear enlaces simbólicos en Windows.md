Abrir una consola como administrador
```
mklink /D C:\LinkToFolder C:\Users\Name\OriginalFolder
```

Referencia: [How to Create Symbolic Links (Symlink) in Windows | Liberian Geek](https://www.liberiangeek.net/2023/10/how-to-create-symbolic-links-symlink-in-windows/)

Nota: Para eliminar un enlace simbólico, simplemente se borra como si se tratara de una carpeta normal.
