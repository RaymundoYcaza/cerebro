- **Ingreso**: Este tipo de partida se utiliza para registrar todas las transacciones que resultan en un aumento de los activos o una disminución de los pasivos de la empresa. Esto podría incluir la venta de bienes o servicios, el recibo de un pago de un cliente, etc.

-

- **Egreso**: Este tipo de partida se utiliza para registrar todas las transacciones que resultan en una disminución de los activos o un aumento de los pasivos de la empresa. Esto podría incluir el pago de gastos, la compra de suministros, el pago a un proveedor, etc.

- **Diario**: Este tipo de partida se utiliza para registrar transacciones que no son ni ingresos ni egresos. Por ejemplo, si transfieres dinero de una cuenta bancaria a otra dentro de tu empresa, podrías usar una partida de diario para registrar la transacción.

- **CxC (Cuentas por Cobrar)**: Este tipo de partida se utiliza para registrar una transacción donde tu empresa ha vendido algo a un cliente y espera recibir el pago en el futuro.

- **CxP (Cuentas por Pagar)**: Este tipo de partida se utiliza para registrar una transacción donde tu empresa ha comprado algo y espera hacer el pago en el futuro.

- **Cierre**: Este tipo de partida se utiliza al final del período contable para transferir los saldos de las cuentas de ingresos, gastos y retiros al resumen de capital. Esto prepara las cuentas para el próximo período contable.
