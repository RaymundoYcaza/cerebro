### 📺 ¿Qué vas a hacer?

1. ¿Necesitas generar contenido creativo o educativo?
   - ¿Es guión, eBook o storytelling?
     → Usa **GPT-4.5** ✅
   - ¿Es contenido técnico o estructurado?
     → Usa **GPT-4o** ✅

2. ¿Vas a trabajar en código o automatización?
   - ¿Es desarrollo PHP, Ionic, React o scripts complejos?
     → Usa **GPT-4.1** ✅
   - ¿Es algo más ligero o rápido?
     → Usa **o4-mini-high** ⚡

3. ¿Estás analizando datos, modelando o diseñando procesos?
   - ¿Es algo que requiere precisión y visualización?
     → Usa **GPT-4o** ✅
   - ¿Quieres profundidad lógica?
     → Usa **o3** 🧠

4. ¿Solo necesitas revisar algo, escribir un email o consultar algo puntual?
   → Usa **GPT-4.1-mini** o **o4-mini** ⚡

5. ¿Quieres la mejor combinación general para productividad, análisis, redacción técnica, visión, programación, etc.?
   → Usa **GPT-4o** 🏆
